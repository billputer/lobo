==========================
Lobo plugin
==========================

Lobo is a Flash plugin for the howl webserver.  Lobo relies on the Flex framework to provide a rich, remote interface for the media player webservice exposed by howl.

For more information, see howl on github: http://github.com/billputer/howl

==========================
Installation Instructions:
==========================

* Import project into Flex Builder 3.  Press the build button.  Copy the compiled .swf file into the plugins/lobo folder of your howl source.
* Alternatively, just use the precompiled version available in howl.
